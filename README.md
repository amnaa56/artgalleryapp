# artgalleryapp
A comprehensive Android application demonstrating authentication, SQLite database, REST API integration, and Material Design.

Features:

User Authentication (Login/Register/Forgot Password)
SQLite Database with CRUD operations
Theme Management (Light/Dark/System)
RecyclerView with Custom Adapter
Options & Context Menus

Technologies Used:
Java
Android SDK
SQLite
Material Design
REST API

Installation:
Clone the repository
Open in Android Studio
Sync Gradle
Run on emulator or device
